# Phase 3 — Backlink & citation outreach targets

Research compiled 2026-07-16 for the AI Readiness implementation plan (Phase 3 — Authority & citation signals). These are candidate directories/partner programs to submit CommerceLead to for backlinks and citation credibility. Submissions and outreach are manual — nothing here has been submitted yet.

## Platform partner directories — highest priority

Direct platform authority, and CommerceLead's claimed Adobe/Shopware certifications make these a natural fit.

| Directory | URL | Notes |
|---|---|---|
| Adobe Solution Partner Directory | https://partners.adobe.com/s/directory/solution | Self-publish listing once at Bronze-level Adobe partner or above. Listings publish within ~2 business days. |
| Magento Association — Members | https://www.magentoassociation.org/members | Individual or company membership; lists Gold/Bronze members. |
| Magento Association — Partners | https://www.magentoassociation.org/partners | Agency-tier partner listing (e.g. Terrific Minds, ZERO-1, DevTeam are current examples). |
| Mage-OS Partner Directory | https://mage-os.org/community/partners/ | Bronze/Silver/Gold/Platinum tiers; logo placement on site + partner pages. Contact info@mage-os.org to start. |
| Shopware Partner Finder | https://www.shopware.com/en/partner/agencies/ | Tiered Bronze→Platinum by certification count (Silver needs 4 Shopware certs, Gold needs 10). |

## General agency/dev marketplaces

Standard B2B service credibility directories — review-dependent, so best after 2-3 real client testimonials exist.

| Directory | URL | Notes |
|---|---|---|
| Clutch.co — Magento developers | https://clutch.co/developers/magento | Requires verified client reviews; also list under https://clutch.co/developers/ecommerce |
| GoodFirms — Magento development | https://www.goodfirms.co/ecommerce-development-companies/magento | Free profile + paid boost tiers. |
| DesignRush — eCommerce agencies | https://www.designrush.com/agency/ecommerce/us | Executive screening process before listing. |
| AgencySlate — Shopware agencies | https://www.agencyslate.com/best-agencies/shopware/all | Merit-based shortlist using partnership status, certifications, reviews. |

## Fractional-CTO-specific directories

Matches the "outsourced CTO" positioning directly. Lower priority — some are paid.

| Directory | URL | Notes |
|---|---|---|
| CTO Directory | https://www.ctodirectory.com/ | Vetted fractional CTO listings. |
| GigX | via https://www.fractionaljobs.io/blog/the-5-best-fractional-cto-platforms-and-services-2026 | Self-service profile, paid monthly (~$89/mo) — evaluate ROI before committing spend. |
| Connectd | https://www.connectd.com/us/fractional-cto-services | Matching-based, free to list, vets both sides. |

## Sequencing

1. Platform directories first (Adobe, Magento Association, Mage-OS, Shopware) — free/certification-gated rather than review-gated, so these can move immediately.
2. Clutch/GoodFirms/DesignRush — hold until 2-3 real client testimonials are ready; a thin profile with zero reviews can look worse than no profile.
3. Fractional CTO directories — optional/paid, lowest priority unless testing that channel specifically for the "Your First CTO" page.
